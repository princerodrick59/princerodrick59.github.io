# FRC Scouting Application

A web-based scouting tool for FIRST Robotics Competition teams to collect match and pit data.

üåê **Live:** [https://princerodrick59.github.io](https://princerodrick59.github.io)

## Features

- Match scouting interface for tracking robot performance
- Pit scouting for documenting robot capabilities
- Automatic data transfer to google sheets
- Works on any device with a browser

## Usage

1. Visit the website
2. Select match or pit scouting
3. Fill out the form
4. Generate and scan QR code to transfer data

## Tech Stack

HTML, CSS, JavaScript, GitHub Pages

## Development

```bash
git clone https://github.com/princerodrick59/princerodrick59.github.io.git
```

Open `index.html` in your browser or deploy via GitHub Pages.
