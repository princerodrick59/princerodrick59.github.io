Key Improvements — 2025 vs 2024
CAN Bus Isolation
CANivore separates drivebase CAN from manipulators. Fixed high utilization + reliability issues from 2024.
Motion Magic
CTRE proprietary profiles replace hand-tuned PID. Smoother mechanism control, dramatically faster tuning.
Fused Encoder Class
Fuses absolute encoder + motor position. Mechanisms work even if absolute encoder fails mid-match.
Global Pose Estimation
PhotonVision AprilTags enables pose estimation in teleop (auto-heading) and autonomous (quicker, more consistent).
Autonomous System

PathPlanner generates trajectories. Sensor fusion (beam breaks + ToF) confirms game piece state at each step — robot waits for confirmed intake before proceeding.

Drive-to-pose in development for teleop — driver presses one button to auto-navigate to reef or human player station, reducing cycle time.
